# 下一步设计：拍照模式 v4（出片闭环）

**Goal:** 在 v3 的基础上补齐“出片闭环”：可微调滤镜强度、可开构图辅助线、保存后给到路径反馈并能一键打开文件夹/复制路径。

---

## 1) 滤镜强度滑杆（Fine Tuning）

在拍照面板中新增滑杆并实时生效（作用于 `PhotoModePostFX`）：
- 柔化（BlurStrength）
- 暗角（VigStrength）
- 饱和（Saturation）
- 对比（Contrast）

额外按钮：
- `恢复默认`：恢复当前滤镜 preset 的默认参数

---

## 2) 构图辅助线（Guides）

新增“辅助线”下拉：
- 无
- 九宫格
- 安全框

实现为 UI 线条覆盖层，仅用于构图参考：
- 拍照保存时会被隐藏（不进照片）

---

## 3) 保存反馈（Save Feedback）

保存成功后弹出轻量 toast：
- 显示文件名/路径（可换行）
- 按钮：
  - `打开文件夹`
  - `复制路径`
  - `关闭`

保存失败也要提示失败信息（最小可用：toast 文本显示失败原因）。

---

## 4) 验收标准

- 滑杆调节实时影响画面，并能恢复默认
- 辅助线可开关，保存时不进图
- 保存后能一键打开文件夹/复制路径
